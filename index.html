ACT 1 · CoCo 기능
Snap and Ask
  • 화면의 어느 영역이든 캡처하여 CoCo에게 자연어로 질문
  • 차트, 대시보드, 오류 메시지, 코드 스크린샷 등 모두 지원
  • CoCo가 시각적 컨텍스트를 완전히 이해하고 분석 결과 제공
  • "이 차트에서 왜 인게이지먼트가 떨어졌나요?" — 자연어 질문
  • 데이터 소스를 자동 추적하여 원인 분석까지 수행
  • 개발자·분석가 모두 사용 가능, 코드 없이도 인사이트 도출
사용 시나리오
  1. Snowsight 대시보드에서 이상한 차트 발견
  2. Snap and Ask로 해당 영역 캡처
  3. "왜 지난주에 비해 30% 하락했나요?" 질문
  4. CoCo가 관련 테이블·파이프라인을 조사해 원인 제시
SE TIP

비개발자 고객(분석가, 경영진)에게 특히 어필되는 기능. "코드 모르셔도 됩니다, 화면 캡처만 하세요"로 포지셔닝. 기존 BI 도구에서 발견한 이상 징후를 즉시 원인 분석할 수 있는 워크플로우를 보여주세요.

ACT 1 · App Development
Snowflake App Runtime & Streamlit
Snowflake App Runtime (Public Preview)
  • Snowflake 내에서 Node.js / React 앱 직접 호스팅

  • Retool, Superblocks 등 외부 앱 빌더와도 통합
Streamlit in Snowflake (GA)
  • 워크스페이스, Git 통합, 컨테이너 실행 환경
  • 월 활성 개발자 1.7M명 — Snowflake 역대 최다
  • Code Bundles (PP): Python 파일을 래퍼 없이 직접 실행
개발자 생태계 확장
  • Excel, VS Code, GitHub Marketplace에서도 CoCo 접근 가능
  • Cortex Function Studio (PP): 맞춤 AI 함수 빌드·평가·배포
  • Agentic Search (PP): 비정형 데이터 → 구조화 → SQL 응답
SE TIP

App Runtime은 Streamlit의 한계(복잡한 UI, 멀티페이지)를 느끼는 고객에게 제안. "Streamlit으로 시작, App Runtime으로 확장" 로드맵을 제시하세요. 1.7M 개발자 수치는 생태계 규모 어필에 효과적.

ACT 2 · 상세
Agent Identity & Horizon Context
Agent Identity
  • 에이전트 실행 컨텍스트를 감지하는 새로운 컨텍스트 함수
  • 마스킹 정책 및 행 접근 정책에서 직접 참조 가능
  • 에이전트 유형별로 다른 데이터 접근 권한 설정
  • "전통적 보안 모델은 인간 사용자를 위해 설계. 에이전트는 다릅니다."
Horizon Context (NEW)
  • CoCo·CoWork·Cortex 에이전트에 시맨틱 신호를 자동 제공
  • Semantic Views와 외부 메타데이터 커넥터 통합
  • 에이전트가 비즈니스 컨텍스트를 이해하고 더 정확한 답변 제공
  • Cortex Sense 연계 → 정확도 24% → 83%
Horizon AI Guardrails
  • CoCo/CoWork를 탈옥 시도로부터 런타임에서 보호
  • 프롬프트 인젝션 공격 방어 (기존 + 제로데이)
SE TIP

Agent Identity는 Cortex Agent를 만드는 고객에게 게임 체인저. 같은 테이블에서 에이전트 유형별로 다른 컬럼을 노출하는 마스킹 정책 데모를 보여주세요. 엔터프라이즈 보안팀의 "AI가 모든 데이터에 접근하면 어쩌나" 우려를 해소합니다.

ACT 2 · Performance
Interactive Compiler & Adaptive Compute
Interactive Compiler (NEW)
  • 새로운 쿼리 컴파일러 도입 — 컴파일 속도 40배 향상
  • 전체 워크로드 3~4배 가속

Adaptive Compute (GA)
  • 1세대 대비 약 2배 빠름, 현재 점진적 롤아웃
  • AI 워크로드용 컴퓨트 리소스 자동 최적화
  • 수동 튜닝 불필요 — Snowflake가 알아서 최적 구성
  • Unistore 8배 최적화 (PP): Hybrid Table 지연·처리량 개선
추가 성능 개선
  • Next-gen Account Replication: 20배 빠름, RPO SLA 보장
  • Observe 인수 완료: 로그·APM·인프라 모니터링 네이티브 통합
  • Streaming Feature Serving GA: 수십 밀리초 지연의 실시간 피처 제공
SE TIP

성능 불만이 있는 고객에게: "40x 컴파일러 개선이 자동 적용됩니다." Adaptive Compute는 "웨어하우스 사이즈 고민 끝"이라는 메시지로. Unistore 고객에게는 8x 개선 소식이 특히 중요합니다.

ACT 3 · Open Sharing
Open Sharing & Data Re-sharing
Open Sharing (Public Preview)
  • Iceberg REST를 통해 비Snowflake 사용자에게 데이터 직접 전달
  • 수신자가 Snowflake 계정 없어도 데이터 접근 가능
  • Spark, Databricks 등 어떤 Iceberg 호환 엔진에서도 읽기 가능

  • Open Semantic Interchange: 54개 벤더 참여 시맨틱 표준
SE TIP

Open Sharing은 "비Snowflake 사용자와도 공유 가능"이라는 강력한 반론. 경쟁사 비교에서 "Snowflake는 닫힌 생태계"라는 공격에 대한 완벽한 응답. Data Re-sharing GA로 마켓플레이스 비즈니스 모델도 가능해집니다.

ACT 3 · Collaboration
Multi-party Collaboration GA
  • 여러 조직이 하나의 안전한 환경에서 데이터를 함께 분석
  • 각 조직의 데이터는 독립적으로 거버넌스 유지 — 상대방에게 원본 미노출
  • Netflix가 얼리 어답터로 실제 운영 중
  • 파트너사·공급사와 Data Clean Room 없이도 프라이버시 보존 분석
  • 마케팅·리서치·금융·헬스케어 등 여러 조직 협업이 필요한 산업에 최적
  • GA 전환으로 엔터프라이즈 SLA 보장
