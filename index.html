1. 편의성 · EASE OF USE
제로 관리 · AI 코딩 에이전트 · 개인 업무 엔진
Snowflake는 정형·반정형·비정형 데이터(JSON, Parquet, Avro, PDF, 이미지, 오디오, 비디오 등)를 단일 플랫폼에서 처리할 수 있는 완전 관리형 클라우드 데이터 플랫폼입니다.
  • 제로 관리: 인덱스 설계, 파티션 튜닝, 용량 산정 등 전통적 DBA 작업이 불필요
  • Adaptive Compute (GA): 워크로드별 최적 컴퓨팅을 자동 배분하여 기존 대비 약 2배 빠른 처리 성능, T-shirt 사이징 불필요
  • 멀티 클라우드: AWS, Azure, GCP 어디서든 동일한 경험 제공
  • Code Bundles (PP): Python/Java 파일을 별도 래핑 없이 Snowflake에서 직접 실행

1. 편의성 · AI AGENTS
Snowflake CoCo & CoWork
Snowflake CoCo (GA)
기존 Cortex Code에서 리브랜딩된 코딩 에이전트
  • 자연어로 데이터 파이프라인 구축, SQL 작성, dbt 빌드, 앱 배포까지 자율 수행
  • Cloud Agents를 통해 로컬 환경 없이도 Snowsight에서 전체 개발 워크플로우 실행

2. 연결성 · CONNECTIVITY
데이터·앱·AI를 하나의 플랫폼에서 실시간 연결
조직 내·외부의 데이터, 애플리케이션, AI 모델을 하나의 플랫폼 위에서 실시간으로 연결하고 공유할 수 있도록 설계되었습니다.
  • Secure Data Sharing: Zero-Copy로 계정 간 즉시 공유 — 지연 없음, 추가 비용 없음
  • Open Sharing (PP): 비Snowflake 조직에도 Iceberg REST Catalog API로 공유
  • Data Resharing (GA): 공유받은 데이터를 제3자에게 재공유
  • Snowflake Marketplace: 3,300개+ 라이브 데이터셋·서비스 즉시 구독
  • Native Apps Framework: Snowflake 위에서 앱을 개발·배포·수익화
  • Apache Iceberg v3 (GA): 삭제 벡터, 행 리니지, Variant 타입 등 가장 광범위한 V3 지원

3. 신뢰성 · TRUST & SECURITY
보안 & 컴플라이언스
엔터프라이즈급 보안, 거버넌스, 옵저버빌리티, 재해 복구 기능을 클라우드·리전에 관계없이 일관되게 제공합니다.
  • 상시 암호화: End-to-End AES-256, Tri-Secret Secure(고객 관리 키)
  • 인증: SOC 2 Type II, HIPAA, FedRAMP, ISO 27001, PCI DSS
  • Dynamic Data Masking, Row Access Policy를 통한 세분화된 접근 제어
  • AI Agent Identity (GA): 모든 AI 에이전트에 암호화 기반 고유 신원 부여, 에이전트별 RBAC, 전체 활동 감사 추적
  • Multi-Party Approval (PP): 최고 위험 작업에 2인 이상 승인 필수화
  • Data Movement Policies (PP): 태그된 데이터의 외부 반출 차단
  • AI Security Posture Management (PP): 프롬프트 인젝션, 탈옥 공격 방어
Agent Identity 핵심
  • 에이전트 실행 컨텍스트 감지
  • 마스킹 정책에서 에이전트 유형별 참조
  • 에이전트별 다른 데이터 접근 권한
Data Movement Policy
  • 외부 스테이지 전송 차단 (S3, Azure Blob)
  • Snowsight 다운로드 차단
  • 에이전트별 이동 정책 차등 적용

3. 신뢰성 · GOVERNANCE
거버넌스 & 옵저버빌리티
  • Horizon Catalog: 데이터 자산 분류, 태깅, 리니지 추적 통합 관리
  • Horizon Context (PP): 외부 메타데이터 수집(PostgreSQL, SQL Server, Tableau, Power BI, dbt), AI 문서 자동 생성, 시맨틱 뷰 자동 탐색
  • Cortex Sense (PP): AI 에이전트가 쿼리 이력·대시보드·메타데이터로부터 런타임에 맥락을 자동 조합하여 정확도 86% 달성 (맥락 없는 모델 단독 24% 대비)
  • Intent-Driven Governance (PP): 자연어로 거버넌스 의도 표현 시 자동으로 분류·정책 적용
    "내 DB의 모든 PII를 보호해줘"

  • Observe by Snowflake: 로그, APM, 인프라 모니터링 네이티브 통합
  • Access History & Query Profile: 누가, 언제, 어떤 데이터에 접근했는지 완전 감사 추적

4. 경제성 · COST EFFICIENCY
사용한 만큼만 과금 · 과잉 프로비저닝 제거
Snowflake는 사용한 만큼만 과금되는 소비 기반(Consumption-Based) 요금 모델을 채택합니다.
  • 컴퓨팅과 스토리지 분리 과금: 쿼리하지 않을 때는 컴퓨팅 비용 $0
  • Adaptive Compute: 워크로드별 최적 리소스 자동 배분, 과잉 프로비저닝 제거
  • 자동 일시 중지/재개: 유휴 시 자동 중지로 불필요한 비용 방지
  • 압축 스토리지: 평균 4~7배 압축으로 저장 비용 절감
  • AI Cost Controls: 기존 예산 체계 연동, 사용자별 할당량, 부서별 비용 귀속, 예산 초과 시 자동 프로시저 실행
  • Serverless 기능 확대: Snowpipe Streaming, Dynamic Tables, Datastream 등 서버리스 기반으로 관리 비용 제거
  • Budget Custom Actions: 지출 임계값 도달 시 저장 프로시저 자동 트리거
