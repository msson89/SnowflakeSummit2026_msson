ACT 1 · STREAMING
Snowflake Data Stream
  • Kafka Wire Protocol compatible — drop-in replacement
  • Fully managed by Snowflake, no cluster ops needed
  • Sub-millisecond latency for real-time use cases
  • Direct integration with Iceberg tables & Snowflake tables
  • Status: Private Preview (expected soon)

For customers running Kafka/Confluent: position Data Stream as operational simplification + cost savings. Key differentiator is Zero-copy to Iceberg tables — no connector maintenance. Calculate TCO savings vs current Kafka + Kafka Connect setup.

ACT 4 · ACQUISITION
